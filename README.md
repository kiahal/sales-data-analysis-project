
# Sales Data Analysis Project 

## Español
Este es un proyecto integral de análisis de datos que abarca desde el modelado conceptual hasta la visualización avanzada. 
Actualmente, el proyecto se encuentra en la etapa de optimización de datos y diseño visual.

### Metodología y Herramientas:
1.  **Modelado de Datos:** Diseño del diagrama Entidad-Relación (DER) en **Draw.io** para definir la estructura y lógica del negocio.
2.  **ETL y Gestión de Datos:** Carga y limpieza manual de registros en **Excel**, utilizando funciones avanzadas y comandos para asegurar la integridad de los datos.
3.  **Visualización y Business Intelligence:**
    *   Diseño de interfaz  funcional en **Canva** para mejorar la experiencia de usuario (UX).
    *   Construcción del dashboard en **Power BI**, optimizando el modelo de datos, relaciones y cardinalidades.
    *   Implementación de lógica de negocio mediante **DAX** para la creación de **KPIs** (Key Performance Indicators) e indicadores clave.
4.  **Análisis mediante SQL**
5.  *   Realice la carga de datos en SQL Server, implemente consultas completas de manera rapida y eficiente.
6.
7.    **Data Storytelling**.
